\section{Security \& Production}
Implemented per \texttt{Plan/SECURITY.md}:
\begin{itemize}[noitemsep]
\item URL validation: block \texttt{file://}, localhost; optional \texttt{BROWSER\_ALLOWED\_HOSTS}
\item JS guardrails: length limit, block \texttt{eval}/storage access patterns
\item \texttt{STALE\_STATE} guard: \texttt{browser\_get\_state} required before click/type
\item Append-only audit log: \texttt{audit.jsonl} under downloads path
\item Navigate retries via \texttt{BROWSER\_NAVIGATE\_RETRIES}
\end{itemize}
CI: GitHub Actions runs \texttt{pytest} on push/PR.
